Now we have some funky radio buttons

When you try to select one, it'll chose a different one instead

This is how I did it:

  1. Create your radio buttons like normal
  2. Add event listeners to each radio button
  3. In the script, use the ".checked" method to code what is checked and what isn't

It's that simple!
